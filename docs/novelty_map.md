# Novelty map (closest neighbors from the systematic search)

BeatState-AF as a *system* is not novel; the morphology+rhythm axis is crowded.
Novelty survives only at the controlled intersection: **capacity-matched state
factorization + complete software-memory ledger + prespecified discordance test**,
framed as an inductive-bias law, not a new SOTA model. Position explicitly against:

- Kim, Lim & Jang — Patient-Adaptive Beat-Wise Temporal Transformer for AF in
  continuous long-term monitoring (IEEE Access, 2024); SeqAFNet (IEEE JBHI, 2024).
- QRS-centric beat-wise AF detection, CNN+BiLSTM (Comput. Biol. Med., 2025).
- HMA-TFN — hierarchical morphology-level + rhythm-level attention (2025).
- Inter-intra period-aware SSL for AF (2024) — factors rhythm vs single-period
  morphology as pretraining tasks.
- MMDN dual-branch; npj-AI rhythm+morphology fusion; RawECGNet; ArNet2.
- Personalized DL for impending-AF forecasting on Icentia11k (arXiv, 2026) —
  forecasting + full fine-tuning, not budgeted detection personalization.

None run the matched-capacity, sparsity-controlled, complete-memory, discordance
experiment. Re-verify this table before claiming novelty (roadmap S01).
